->Design Principles
The redesign was driven by clarity, usability, and visual hierarchy. I focused on making key user actions—such as browsing and registering for workshops—immediately accessible. A card-based layout, consistent spacing system, and clear typography were introduced to reduce cognitive load and improve content scannability. Consistency across components ensures a predictable and intuitive user experience.

->Responsiveness
I followed a mobile-first design strategy, considering that most users access the platform on mobile devices. Layouts were built using Flexbox and CSS Grid to adapt fluidly across screen sizes. Special attention was given to touch targets, readable font scaling, and vertical content flow, ensuring a seamless experience from mobile to desktop.

->Trade-offs
A key trade-off was balancing visual enhancement with performance. Instead of using heavy UI libraries or complex animations, I opted for lightweight styling and subtle interactions. This approach ensures faster load times and smoother performance, particularly on low-bandwidth networks and lower-end devices, while still delivering a modern interface.

->Challenges & Approach
The most challenging aspect was improving the user interface without modifying the underlying functionality. To address this, I adopted a component-driven approach, isolating UI improvements into reusable elements and iteratively refining them. Continuous testing across screen sizes helped maintain both functional integrity and responsiveness while enhancing the overall user experience.
